# Simple Client-Server Chat (Python)

Простое консольное приложение для обмена сообщениями между клиентом и сервером через сокеты.

## Возможности
- Двусторонний обмен сообщениями
- Работа в реальном времени
- Завершение соединения по команде `[STOP]`

## Требования
- Python 3.x

## Запуск

### 1. Запуск сервера
```bash
python server.py

### 2. Запуск сервера
```bash
python client.py
